About Jade | Jadeとは

JadeはHamlに大きな影響を受けたnode向けのハイパフォーマンスなテンプレートエンジンで、JavaScriptによって実装されています。
- Jade
- Jade on github

Features | 主な機能

  • クライアントサイドのサポート
  • 高い可読性
  • アトリビュートの補間
  • 柔軟なインデント構文
  • 拡張ブロック
  • セキュリティに考慮し、デフォルトでコードをエスケープ
  • コンパイル＆実行時の構文エラー報告
  • コマンドラインからjadeテンプレートをコンパイルして実行可能
  • HTML5モード(DOCTYPEとして!!! 5を使用)
  • メモリキャッシュ(オプション)
  • 動的/静的なタグクラスの組み合わせ
  • 「フィルタ」を使用した構文木の操作
  • Expressの標準サポート
  • オブジェクト、配列、あるいは列挙に対応していないデータの- eachによるイテレーション
  • ブロックコメント
  • タグプリフィクス無し
  • AST(Abstract Syntax Tree, 抽象構文木)フィルタ
  • フィルタ -
    • :sass sass.jsのインストールで使用可能
    • :less less.jsのインストールで使用可能
    • :markdown markdown-jsまたはnode-discountのインストールで使用可能
    • :cdata
    • :coffeescript coffee-scriptのインストールで使用可能
  • Vimシンタックス
  • TextMateバンドル
  • Screencasts
  • html2jadeを使用した変換

Implementations | 実装されている言語

  • PHP - jade.php
  • Scala - Scaml
  • Ruby - Slim

Public API | パブリックAPI

                            var jade = require('jade');
                            
                            // 文字列をレンダリング
                            jade.render('string of jade', { options: 'here' });
                            
                            // ファイルの内容をレンダリング
                            jade.renderFile('path/to/some.jade', { options: 'here' }, function(err, html) {
                                // "options"を省略して、コールバックを第2引数にすることも出来る.
                            });
                            
                            // 関数をコンパイル.
                            var fn = jade.compile('string of jade', options);
                            fn.call(scope, locals);

オプション

  • scope: スコープ(this)を設定します。
  • self: ローカル変数の束縛にself名前空間を使用します。デフォルトではfalseです。
  • locals: ローカル変数を含んだオブジェクト。
  • filename: 例外発生時に使用されるファイル。また、cacheの設定時にこのファイルがrequireされます。
  • cache: Cache intermediate JavaScript in memory keyed by filename
  • debug: Outputs tokens and function body generated
  • compiler: デフォルトのコンパイラを置き換えます。
